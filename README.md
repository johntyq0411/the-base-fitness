# Modern Fitness Kluang Gym Website

This is a premium, highly responsive, and reusable gym website built with React, Vite, and Vanilla CSS.

## Features
- **Hero & Landing Page**: Modern, high-energy layout tailored for fitness and strength training.
- **Membership & Class Booking**: Register accounts, purchase subscriptions, and book group timetable slots.
- **Personal Trainer Portal**: Coaches can view assigned members and training slots.
- **Admin Configuration Panel**: Gym managers can edit packages, schedules, and branding color themes.
